# 행렬계산기

[구현 사이트 이동](https://jyeon-matrix.netlify.app/)

1x1 ~ 6x6 행렬을 제작하고 더하기, 빼기, 곱하기 계산을 할 수 있습니다.

행렬 제작 후 오른쪽 중앙에 있는 random 숫자 버튼을 클릭하면 각 칸에 랜덤으로 숫자가 채워집니다.

(직접 입력도 가능합니다.)
